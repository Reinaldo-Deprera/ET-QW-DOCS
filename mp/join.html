PLAY ONLINE

NOTE: A BROADBAND INTERNET CONNECTION IS REQUIRED (E.G., DSL, CABLE MODEM, ETC.).

Clicking “Play Online” gets you the in game browser seen below:

  1. The drop down menu here lists where the list of games is coming from. You can choose from Internet, LAN (Local Area Network), or History.
  2. This is where servers will be listed. You can sort by server name, map name, game type, player count, and ping. Double-click a server to join it.
  3. Toggle between Friends list and Clan list here.
  4. This is where your friends/clan will be listed. You can also add to either list by clicking the plus sign and entering the player’s name you wish to add. You can also create a clan here. Right click within each list for additional functionality.
  5. Direct Connect to a server by entering its IP addresses, and protocol.
  6. Refresh the current list.
  7. Customize your server filters here. Specify variables like player count or game type to only list those you want to view in the browser.
  8. Displays the Server Information of the currently selected server.
