There are six primary Objectives in Enemy Territory: QUAKE Wars

Destroy

GDF Soldier/Strogg Aggressor only.
Map examples include the Stroggifier in Valley, the Jamming Generator in Area 22 and Ark.



Escort MCP

Any GDF Class can drive the MCP.
Map examples include Valley, Outskirts and Slipgate.




Construction

GDF Engineer/Strogg Constructor only.
Map examples include the Bridges in Canyon and Valley, the Mining Laser in Area22 and Ark, the EMP Disruptor in Sewer.



Hack

GDF Covert Ops/Strogg Infiltrator only
Map examples include the Sewer Controls in Sewer, the Slipgate in Slipgate




Transport

Any Class not in a vehicle can complete Transport Objectives.
Map examples include the DataBrain in Outskirts, the DataDisc in Island, and the Energy Cells in Quarry.



Flyer Drone Attack

Strogg Infiltrator Only
Map example Volcano only
